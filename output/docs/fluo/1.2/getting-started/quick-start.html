Quick Start

Edit this page

Follow the steps below to run a Fluo application:

  1. Install Fluo

  2. Create a Fluo application or use an example application.

  3. Initialize your Fluo application

  4. Pick a method below to run your Fluo application

    • Run Fluo processes
    • Run Fluo in YARN
    • Run Fluo in Docker which enables running in Mesos and Kubernetes
